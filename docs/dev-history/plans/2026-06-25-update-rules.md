## Background context for all tasks

### Key constants / paths
- `rules/sigma/` — encoded rules directory (what gets updated)
- `rules/ruleset.toml` — rule subset list (operator-editable, committed)
- `rules/sigma/PROVENANCE` — pin file read by `cairn_sigma::ruleset::read_pin()`
- SSRF base URL: `"https://raw.githubusercontent.com/SigmaHQ/sigma/"`
- XOR key: `cairn_sigma::codec::KEY` = `b"cairn-rules-v1-not-a-secret"` (public)

### Existing crates to depend on
- `cairn-core`: `CairnError`, `Result`
- `cairn-sigma`: `codec::xor(bytes) -> Vec<u8>`

### PROVENANCE file format (must match what `cairn_sigma::ruleset::read_pin` expects)
```
# Bundled Sigma ruleset provenance (ADR-0003).
# Regenerated by cairn update-rules. The `pin` is the upstream SigmaHQ commit
# this XOR-encoded bundle was fetched from.
pin = <40-hex-sha>
source = https://github.com/SigmaHQ/sigma
```
`read_pin` looks for a line containing `pin =` and takes the value after `=`.

### DRL 1.1 validation
Each fetched `.yml` must contain at least one line starting with `author:` (bytes check,
no YAML parsing needed). If absent → `CairnError::Other("DRL 1.1: missing author in ...")`.

### SSRF protection
Pin must match `^[0-9a-f]{40}$`. URL must start with
`"https://raw.githubusercontent.com/SigmaHQ/sigma/"`. Any other URL → error before fetch.

#![forbid(unsafe_code)]

use cairn_core::{CairnError, Result};

const BASE: &str = "https://raw.githubusercontent.com/SigmaHQ/sigma/";

fn validate_pin(pin: &str) -> Result<()> {
    if pin.len() == 40 && pin.bytes().all(|b| b.is_ascii_hexdigit() && !b.is_ascii_uppercase()) {
        Ok(())
    } else {
        Err(CairnError::Other(format!(
            "invalid pin {pin:?}: must be exactly 40 lowercase hex chars (a-f0-9)"
        )))
    }
}

fn build_url(pin: &str, rule_path: &str) -> Result<String> {
    validate_pin(pin)?;
    let url = format!("{BASE}{pin}/rules/{rule_path}");
    // SSRF gate: assert the constructed URL starts with the hardcoded base.
    // This is logically guaranteed by the format! above but we assert defensively.
    if !url.starts_with(BASE) {
        return Err(CairnError::Other(format!("SSRF gate: URL does not start with expected base: {url}")));
    }
    Ok(url)
}

fn check_drl11(bytes: &[u8], rule_path: &str) -> Result<()> {
    let has_author = bytes
        .split(|&b| b == b'\n')
        .any(|line| line.starts_with(b"author:"));
    if has_author {
        Ok(())
    } else {
        Err(CairnError::Other(format!(
            "DRL 1.1 violation: {rule_path} has no `author:` field"
        )))
    }
}

pub fn fetch_rule(pin: &str, rule_path: &str) -> Result<Vec<u8>> {
    let url = build_url(pin, rule_path)?;
    let response = reqwest::blocking::get(&url)
        .map_err(|e| CairnError::Other(format!("fetch {rule_path}: {e}")))?;
    if !response.status().is_success() {
        return Err(CairnError::Other(format!(
            "fetch {rule_path}: HTTP {} from {url}",
            response.status()
        )));
    }
    let bytes = response.bytes()
        .map_err(|e| CairnError::Other(format!("read body {rule_path}: {e}")))?
        .to_vec();
    check_drl11(&bytes, rule_path)?;
    Ok(bytes)
}
```

pub fn run(
    pin_override: Option<&str>,
    rules_dir: &Path,
    ruleset_toml: &Path,
) -> Result<()> {
    let cfg = config::load(ruleset_toml)?;
    let pin = pin_override.unwrap_or(&cfg.sigma.pin);

    // SSRF gate: validate pin before any network access.
    fetch::validate_pin_pub(pin)?;

    tracing::info!(pin, rules = cfg.rules.len(), "update-rules: fetching");

    for rule in &cfg.rules {
        let filename = std::path::Path::new(&rule.path)
            .file_name()
            .and_then(|n| n.to_str())
            .ok_or_else(|| cairn_core::CairnError::Other(
                format!("invalid rule path: {}", rule.path)
            ))?
            .to_string();

        tracing::info!(path = %rule.path, "fetching");
        let plain = fetch::fetch_rule(pin, &rule.path)?;
        encode::write_encoded_rule(rules_dir, &filename, &plain)?;
        tracing::info!(file = %filename, "written (XOR-encoded)");
    }

    encode::write_provenance(rules_dir, pin)?;
    tracing::info!(pin, "PROVENANCE updated");

    // Post-write integrity: compute aggregate and log it.
    let ver = cairn_sigma::ruleset::ruleset_version(rules_dir, false)?;
    tracing::info!(sigma_ruleset_ver = %ver, "update-rules complete");

    Ok(())
}
```
